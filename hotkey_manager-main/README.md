# Hotkey Manager 🖥️
이 프로그램은 **설치된 앱들의 핫키를 정리하고 관리**할 수 있도록 도와줍니다.

## 🔹 사용 방법
1. 프로그램을 실행합니다.
2. `앱 이름`과 `핫키`를 입력한 후 **"추가" 버튼**을 클릭하면 테이블에 추가됩니다.
3. 이후 핫키 정보를 쉽게 관리할 수 있습니다.
